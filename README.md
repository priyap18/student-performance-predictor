# 📚 Student Performance Predictor

A Machine Learning web application built using **Flask** and **Multiple Linear Regression** that predicts a student's performance score based on various academic and lifestyle factors.

---

## 🚀 Project Overview

This application predicts a student's performance index using the following features:

* Hours Studied
* Previous Scores
* Extracurricular Activities
* Sleep Hours
* Sample Question Papers Practiced

The model is trained using **Multiple Linear Regression** from Scikit-Learn and deployed through a Flask web application.

## 📊 Input Features

| Feature                    | Description                                 |
| -------------------------- | ------------------------------------------- |
| Hours Studied              | Number of hours studied per day             |
| Previous Scores            | Previous examination score                  |
| Extracurricular Activities | Participation in extracurricular activities |
| Sleep Hours                | Average sleep hours per day                 |
| Sample Papers Practiced    | Number of sample papers solved              |

---

## 🎯 Output

The model predicts the student's **Performance Index / Exam Score** based on the provided inputs.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/priyap18/student-performance-predictor.git
```

### Move to Project Folder

```bash
cd student-performance-predictor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
